Detailed Description

Helper for building a list of parameters for calling a Proxy_Function it does automatic conversion to Boxed_Value types via operator<<

example usage: Boxed_Value retval = dispatch(dispatchengine.get_function("+"), chaiscript::Param_List_Builder() << 5 << 6);
