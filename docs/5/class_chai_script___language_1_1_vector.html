Detailed Description

A vector of Objects.

ChaiScript includes a shortcut for creating a Vector of Objects

Example:

eval> var v = [1,2,3,4]
[1, 2, 3, 4]
eval> v[0];
1
eval> v.size();
4

Implemented with std::vector<chaiscript::Boxed_Value>
